This is an implementation of Head pose estimation algorithm from single depth Kinect 2 frame.
More detailed information could be found at 
http://fsukno.atspace.eu/Research.htm#HeadPoseChall17

=======================
The algorithms are available as Matlab code:

Main function: HeadPoseEstim_CMTech.m
input:         npy file name with depth information from Kinect 2
output:        pitch, yaw, roll angles
